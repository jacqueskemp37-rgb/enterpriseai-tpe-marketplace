# EnterpriseAI TPE — Claude Cowork Plugin

**The Complete, Consistent & Comprehensive (3C) Business Planning Framework**

## What Is This?

This is a Claude Cowork plugin that turns Claude into an expert business
planning agent using the EnterpriseAI TPE (Think-Plan-Execute) methodology.

Unlike existing Cowork plugins that serve individual departments (Sales,
Finance, HR), this plugin addresses **whole-firm business planning** —
connecting all departments into one unified, living business plan.

## Plugin Contents

### 7 Skill Files (skills/)
| Skill | Description |
|-------|-------------|
| tpe-framework.md | Core methodology: 3 layers, 6 dimensions, alignment rules |
| portfolio-driver.md | Business lines, M&A, partnerships, growth strategy |
| marketing-driver.md | Customers, products, channels, go-to-market |
| organizational-driver.md | Org structure, HR, culture, governance |
| operational-driver.md | Processes, IT, customer service, automation |
| reputation-driver.md | Brand, communications, legal, compliance |
| financial-driver.md | Budgets, KPIs, risk, performance (integration driver) |

### 8 Slash Commands (commands/)
| Command | Description | Output |
|---------|-------------|--------|
| /tpe:full-plan | Complete business plan creation | Word + Excel + PPTX |
| /tpe:driver-audit | Score drivers against TPE standards | Gap analysis report |
| /tpe:alignment-check | Cross-check plans for consistency | Alignment matrix |
| /tpe:connect-the-dots | Map interdependencies | Dependency map |
| /tpe:quick-scan | Rapid gap assessment (free tier) | 1-page RAG summary |
| /tpe:kpi-dashboard | Build KPI tracking dashboard | Excel workbook |
| /tpe:onboard | TPE training for new team members | Training deck (PPTX) |
| /tpe:scheduled-review | Recurring plan reviews | Review agenda + dashboard |

### Connectors (.mcp.json)
Google Drive, Excel, PowerPoint, Gmail, Slack, Calendar

### Sub-Agents (sub-agents/)
Parallel Driver Analysis — analyze all 6 drivers simultaneously

## Quick Start

1. Run `/tpe:quick-scan` to assess your current planning maturity
2. Review the RAG-rated results with your leadership team
3. Run `/tpe:full-plan` to build your complete business plan
4. Set up `/tpe:scheduled-review` for ongoing plan maintenance
